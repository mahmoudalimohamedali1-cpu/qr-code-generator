# QR Code Generator

A modern, responsive web application for generating QR codes from user input.

## Features

- Clean, modern UI with gradient background
- Responsive design for desktop and mobile
- Form validation
- Generate QR codes from:
  - Title (optional)
  - URL or Data (required)
  - Note (optional)
- Download QR code as PNG
- Copy encoded data to clipboard
- Clear form functionality

## Usage

Simply open `index.html` in your web browser. No build process or dependencies required (library loaded from CDN).

## Technologies

- HTML5
- CSS3
- JavaScript (Vanilla)
- QRCode.js library (via CDN)
